

## תוכנית: דיאלוג בחירת יעד להתראה בערוצים מרובי משתמשים

### מה ישתנה

כשלוחצים על כפתור הפעמון (🔔) בהודעה:
- **ערוץ עם משתמש אחד בלבד** → שליחה ישירה כמו היום (ללא דיאלוג)
- **ערוץ עם משתמשים מרובים** → נפתח דיאלוג עם שתי אפשרויות:
  1. **שלח לקבוצה משותפת** (ברירת מחדל, מסומנת) — שולח לקבוצת הווטסאפ המשויכת לערוץ (`notification_group_link`)
  2. **שלח לאיש קשר ספציפי** — בחירת חבר ערוץ מרשימה, שליחה למספר הטלפון שלו

### שינויים טכניים

**1. קומפוננטת `NotifyTargetDialog` חדשה (בתוך `TeamChat.tsx`)**
- RadioGroup עם שתי אפשרויות: "קבוצה" / "איש קשר"
- כשנבחר "איש קשר" — dropdown עם חברי הערוץ (שם + טלפון)
- כפתור "שלח התראה"

**2. עדכון Edge Function `notify-team-message`**
- קבלת פרמטר חדש `targetOverride`: `{ type: 'group' }` או `{ type: 'contact', phone: '...', name: '...' }`
- כש-`targetOverride` קיים — דריסת הלוגיקה הקיימת ושליחה רק ליעד שנבחר
- כשאין `targetOverride` — ההתנהגות הקיימת נשמרת

**3. עדכון זרימת הלחיצה ב-`TeamChat.tsx`**
- `onNotifyMessage` → בודק אם יש יותר מחבר אחד → פותח דיאלוג / שולח ישירות
