

# ייבוא לקוחות מ-CSV עם מיפוי שירותים וחשבונות מודעות

## סיכום
סקריפט Python שיייבא 103 לקוחות מהקובץ שהועלה ישירות לבסיס הנתונים, כאשר:
- **שירותים** — יומרו מטקסט מופרד בפסיקים (`seo,ppc_google`) למערך PostgreSQL (`{seo,ppc_google}`)
- **חשבונות מודעות** — ינוקו מערכים כמו "לא רלוונטי", "יש לך" (יישמרו כ-NULL)
- **טלפונים** — ינוקו מתווי RTL ותבנית +972
- **קמפיינרים** — ישויכו ב-`client_team` לפי שמות

## שלבים

### 1. סקריפט Python לייבוא
- קריאת ה-CSV עם pandas
- ניקוי שדות: טלפון (הסרת `⁦⁩`, `+972`, מקפים), אימייל (הסרת RTL markers)
- המרת `services` מ-`"seo,ppc_google"` ל-PostgreSQL array `{seo,ppc_google}`
- סינון חשבונות מודעות: "לא רלוונטי" / "יש לך" / ריק → NULL
- `contact_name` מעמודת "איש קשר"
- INSERT ישירות דרך `psql`

### 2. שיוך קמפיינרים
- פירוק עמודת "קמפיינר (מקורי)" לשמות בודדים (פיצול לפי `,`, `+`, `ו`)
- התאמה לרשימת הקמפיינרים הקיימת בבסיס הנתונים
- הכנסה ל-`client_team` עם `role = 'primary'` לראשון ו-`'secondary'` לשאר

### 3. וולידציה
- בדיקה שכל 103 הלקוחות נכנסו
- אימות שהשירותים נשמרו כמערכים תקינים
