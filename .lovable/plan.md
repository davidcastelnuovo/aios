

# תוכנית: חסימה אוטומטית של כל השולחים עם אותו נושא

## הבעיה
כשחוסמים שולח, רק כתובת המייל הספציפית נחסמת. אימיילים עם אותו נושא משולחים אחרים ממשיכים להופיע.

## הפתרון
בפונקציית `blockSender`, אחרי חסימת השולח הנוכחי:
1. לחפש ב-Gmail API את כל האימיילים עם אותו נושא בדיוק (`subject:"..."`, עד 500 תוצאות)
2. לחלץ את כתובת המייל של כל שולח
3. לבצע `insert` ל-`gmail_blocked_senders` עבור כל כתובת ייחודית שנמצאה

## שינויים

### `src/pages/Gmail.tsx` — פונקציית `blockSender`
- לקבל את ה-`messageId` בנוסף לאימייל (או למצוא את ההודעה מהרשימה)
- לשלוף את נושא ההודעה
- לקרוא ל-`gmail-api` עם `action: 'list'` ו-`query: subject:"הנושא"`
- לקרוא ל-`gmail-api` עם `action: 'get'` לכל הודעה כדי לחלץ את כתובת השולח
- לחסום את כל הכתובות הייחודיות שנמצאו

שינוי בקובץ אחד בלבד.
