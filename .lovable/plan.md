## הבעיה

שורת הטאבים ב-`AgentEditor` כוללת 13 טאבים שנשפכים לשתי שורות (פרופיל, מטרות, משימות, כלים, מאגר כלים, MCP, Supervisor, ריצות, Evals, ידע, זיכרון, אישורים, פרופילי משתמשים, עלות). עומס ויזואלי, וקשה למצוא דברים.

בנוסף, אין גישה ישירה לעמוד ניהול משימות הסוכנים (`/agent-tasks`) שמראה את כל המשימות ומצבן (pending / running / completed וכו').

## הפתרון

לצמצם את שורת הטאבים לשני אלמנטים בלבד:

```text
[ ⚙️ הגדרות סוכן ▾ ]   [ 📋 משימות סוכן ]
```

### 1. Dropdown "הגדרות סוכן"

תפריט אחד (Popover או DropdownMenu של shadcn) שמרכז את כל הטאבים הקיימים פרט ל"משימות":
פרופיל · מטרות · כלים · מאגר כלים · MCP · Supervisor · ריצות · Evals · ידע · זיכרון · אישורים · פרופילי משתמשים · עלות.

- בחירה בפריט מעדכנת את ה-`tab` query param (כמו היום) ומציגה את אותו `TabsContent` שכבר קיים.
- ה-label של הכפתור מציג את שם הפריט הפעיל ("הגדרות סוכן · ריצות") כדי שתמיד יהיה ברור איפה אנחנו.
- ניתן לקבץ בתפריט בקטגוריות עם separators:
  - **תצורה:** פרופיל, מטרות, כלים, מאגר כלים, MCP, Supervisor
  - **תפעול וניטור:** ריצות, Evals, אישורים, עלות
  - **ידע ופרסונליזציה:** ידע, זיכרון, פרופילי משתמשים

### 2. כפתור "משימות סוכן"

כפתור בולט ליד הדרופדאון (לא חלק מה-Tabs) שמנווט ל-`/agent-tasks` (העמוד הקיים `AgentTasksPage`). זה עמוד הניהול המלא — רואים את כל המשימות, סטטוס (pending/running/completed/failed), תיזמון, היסטוריה, וניתן להריץ ידנית.

הטאב הישן "📋 משימות" (`TasksTab` הפנים-סוכני) **נשאר** כפריט בתוך הדרופדאון תחת הקטגוריה "תפעול וניטור" — הוא ייעודי לסוכן הנוכחי בלבד. הכפתור החיצוני "משימות סוכן" מוביל לעמוד הגלובלי על פני כל הסוכנים.

## קבצים לשינוי

**`src/components/agents/AgentEditor.tsx`** (שורות 89-104):

- להחליף את `<TabsList>` הרחב בשורה צרה: `DropdownMenu` (shadcn) משמאל, `Button` עם `navigate("/agent-tasks")` מימין.
- ה-`<Tabs>` והכל `<TabsContent>` נשארים כפי שהם — רק ה-Trigger UI משתנה. הערך הנבחר מנוהל כבר דרך ה-URL param.
- להוסיף קבוע אחד עם מערך פריטי התפריט (id, label, icon, group) כדי להימנע מ-13 שורות JSX.

**אין שינויים** ב:
- `TasksTab.tsx`, `RunsTab.tsx` וכל יתר ה-tabs — הם נשארים כמו שהם.
- `AgentTasksPage.tsx` — קיים כבר ועובד, רק נחשוף קישור אליו.
- ראוטים — `/agent-tasks` כבר רשום ב-`App.tsx:225`.

## אחרי השינוי

שורה אחת נקייה במקום שתיים. גישה בלחיצה אחת לעמוד ניהול המשימות הגלובלי, כל יתר הטאבים נגישים מתפריט מקובץ.
