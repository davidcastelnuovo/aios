

# תצוגת צ'אט ללקוחות (ClientsChatView) - בדיוק כמו בלידים

## מה נבנה
רכיב `ClientsChatView` חדש בסגנון WhatsApp, זהה לתצוגת הצ'אט של הלידים (`LeadsChatView`), שמציג רשימת לקוחות בצד ימין (25%) ופאנל פרטים בצד שמאל (75%).

## שינויים

### 1. רכיב חדש: `src/components/clients/ClientsChatView.tsx`
- רשימת לקוחות בצד ימין עם חיפוש, multi-select, ובחירה מרובה למחיקה/עדכון סטטוס/עדכון מצב רוח
- פאנל פרטים בצד שמאל עם:
  - **סרגל כלים**: שם הלקוח, סטטוס (פעיל/מושהה/הסתיים/בקליטה), מצב רוח (emoji), טלפון, אימייל, עריכה, מחיקה
  - **טאבים**: פרטי לקוח | משימות ועדכונים | WhatsApp
- כל פריט ברשימה יציג: שם לקוח, שם סוכנות, סטטוס (badge צבעוני), מצב רוח (emoji)
- Multi-select עם פעולות: שינוי סטטוס, שינוי מצב רוח, מחיקה

### 2. עדכון: `src/pages/Clients.tsx`
- הוספת viewMode שלישי `"chat"` (ליד grid ו-table)
- כפתור חדש בסרגל הכלים לתצוגת צ'אט (אייקון MessageCircle)
- כשנבחר viewMode=chat, מציג את `ClientsChatView` במקום הגריד/טבלה

### פרטי הטאבים בפאנל הפרטים
- **פרטי לקוח**: שם, סוכנות, איש קשר, טלפון, אימייל, אתר, ריטיינר, תקציב חודשי, תעשייה, הערות, תאריך התחלה
- **משימות ועדכונים**: שימוש ב-`ClientUpdatesTab` הקיים
- **WhatsApp**: שימוש ב-`ChatViewComponent` הקיים (כמו בלידים)

### Query Keys לעדכון אחרי פעולות bulk
- `["clients"]` - המפתח הקיים שכבר משמש בדף הלקוחות
