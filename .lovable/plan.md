

## תיקון Layout מובייל של עמוד משימות

### הבעיה
במובייל, רשימת המשימות (TaskBacklogPanel) צרה ודחוסה בצד, והיומן מציג מספר ימים בגלילה אופקית במקום יום אחד ברוחב מלא.

### הפתרון

**קובץ: `src/components/tasks/WeeklyTaskBoard.tsx`** — שינוי ה-Mobile Layout (שורות 1020-1157):

1. **רשימת משימות ברוחב מלא** — הפאנל יתפוס את כל רוחב המסך, עם `max-h-[45vh]` וגלילה פנימית כדי לא לתפוס יותר מחצי גובה העמוד
2. **הפרדת Header מהפאנל** — הכותרת, ניווט ופילטרים ישארו מעל הפאנל, הפאנל מתחתיהם
3. **יומן יום אחד ברוחב מלא** — בתצוגה שבועית במובייל, להציג רק את היום הנוכחי (`currentDate`) במקום כל הימים, ברוחב מלא ללא גלילה אופקית

**קובץ: `src/components/tasks/OverdueTasksPanel.tsx`** — הוספת prop או class לרוחב מלא במובייל (אם הפאנל מגביל רוחב בעצמו)

### מבנה מובייל חדש
```text
┌─────────────────────────┐
│  כותרת + ניווט + פילטרים │
├─────────────────────────┤
│  רשימת משימות (full-width)│
│  max 45vh + scroll       │
├─────────────────────────┤
│  יומן - יום אחד בלבד    │
│  full-width              │
└─────────────────────────┘
```
