

# תיקון שמות ארוכים שדוחפים תוכן מחוץ לכרטיסי לידים

## הבעיה
שמות ארוכים (שם איש קשר או שם חברה) גורמים לכרטיס הליד להתרחב מעבר לרוחב העמודה בקנבן, מה שדוחף את שאר הכרטיסים ומשבש את הלייאוט.

## הפתרון
שלושה תיקוני CSS בקובץ `src/pages/Leads.tsx`:

1. **כרטיס הליד (Card)** — הוספת `overflow-hidden min-w-0` כדי שהכרטיס לא יחרוג מגבולות העמודה
2. **שם חברה (שורה 271)** — הוספת `truncate` לטקסט שם החברה כך שייחתך עם שלוש נקודות במקום לדחוף החוצה
3. **עמודת הקנבן (שורה 2669)** — הוספת `min-w-0 overflow-hidden` לכל עמודת שלב כדי להבטיח שהגריד לא יישבר

## פרטים טכניים
- שורה 248: `className="mb-3 hover:shadow-lg transition-all"` → הוספת `overflow-hidden min-w-0`
- שורה 271: `<span className="text-xs text-muted-foreground">` → הוספת `truncate`
- שורה 2671: `className="relative"` → הוספת `min-w-0 overflow-hidden`
