

# תוכנית: הוספת "מעקב דירוגים" לתפריט

## סקירה
נוסיף קישור ל-**מעקב דירוגים (Rank Tracking)** בתפריט הצד, מתחת לפריט **"אנליטיקס אתרים"** הקיים.

## מה ייעשה

### 1. הוספת פריט לטבלת menu_items
נריץ מיגרציה שתוסיף את פריט התפריט החדש לכל ה-tenants הקיימים:

| שדה | ערך |
|-----|-----|
| menu_key | `rank_tracking` |
| original_label | `מעקב מיקומים` |
| route | `/rank-tracking` |
| icon | `Target` |
| parent_menu_key | `management` |
| sort_order | `107` (מיד אחרי site_analytics) |
| category | `management` |

### 2. עדכון מיפוי הרשאות
נוסיף את הפריט החדש למיפוי ההרשאות ב-`AppSidebar.tsx`:

```text
'rank_tracking': 'clients'
```

כך משתמשים עם הרשאת גישה ללקוחות יוכלו לראות גם את מעקב הדירוגים.

## קבצים שיעודכנו

| קובץ | שינוי |
|------|-------|
| מיגרציית Database | הוספת פריט rank_tracking לכל tenants |
| `src/components/layout/AppSidebar.tsx` | הוספת מיפוי הרשאות |

## תוצאה צפויה
בתפריט הצד, תחת קטגוריית **ניהול**, יופיע:
- אנליטיקס אתרים
- **מעקב מיקומים** (חדש)
