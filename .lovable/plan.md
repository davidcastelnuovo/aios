

# עיצוב מחדש של עמוד Gmail - בסגנון Gmail אמיתי

## הבעיות הנוכחיות
1. רשימת ההודעות לא דומה לג'ימייל — חסרות שעות, שם השולח לא בולט, אין הפרדה ברורה בין נושא לתקציר
2. אין סינון לפי תאריך ואין ניווט קדימה/אחורה בין דפים
3. כל ההודעות נמשכות בבת אחת — צריך לתמוך בשליפה לפי טווח תאריכים

## שינויים ב-Frontend (`src/pages/Gmail.tsx`)

### 1. עיצוב שורת הודעה בסגנון Gmail
כל שורת הודעה תהיה בשורה אחת (row) עם:
- **ימין**: שם השולח (bold אם unread), רוחב קבוע ~180px
- **אמצע**: נושא ההודעה + תקציר (snippet) בצבע אפור, חתוך ב-truncate
- **שמאל**: שעה/תאריך

### 2. סינון לפי תאריך
- הוספת Date picker עם בחירת יום ספציפי
- כפתורי חצים לניווט יום קדימה/אחורה
- כשמשנים תאריך — שליחת query לפי `after:YYYY/MM/DD before:YYYY/MM/DD` ל-Gmail API

### 3. פגינציה (קדימה/אחורה)
- שמירת מערך של pageTokens שכבר ביקרנו בהם
- כפתור "הבא" — שולף עם nextPageToken
- כפתור "הקודם" — חוזר ל-token הקודם מהמערך
- מציג "עמוד X" במרכז

### 4. שליפה לפי תאריך בלבד (לא סנכרון הכל)
- ה-Gmail API כבר תומך ב-query parameter `q` עם `after:` ו-`before:`
- אין צורך בשינוי ב-Edge Function — רק שליחת ה-query הנכון מהצד לקוח

## שינויים ב-Backend
**אין שינויים נדרשים** — ה-Edge Function כבר תומך ב-`query` parameter שנשלח ל-Gmail API.

## פירוט טכני

### מבנה שורת הודעה חדש
```text
┌──────────────────────────────────────────────────────────────┐
│  13:39  │  נושא ההודעה - תקציר קצר באפור...  │  שם השולח  │
│  (שמאל) │          (אמצע, flex-1)              │   (ימין)   │
└──────────────────────────────────────────────────────────────┘
```

### סרגל סינון חדש
```text
┌──────────────────────────────────────────────────────────────┐
│  [מייל חדש] [⚙]  │  [◄] 7 מרץ 2026 [►]  │  [🔍 חיפוש]  │
│                   │  בורר תאריך           │               │
└──────────────────────────────────────────────────────────────┘
```

### פגינציה
```text
┌──────────────────────────────────────────────────────────────┐
│  [→ הקודם]        עמוד 1          [הבא ←]                  │
└──────────────────────────────────────────────────────────────┘
```

### State חדש
- `selectedDate: Date` — תאריך נבחר (ברירת מחדל: היום)
- `pageTokenHistory: string[]` — מערך tokens לניווט אחורה
- `currentPage: number` — מספר עמוד נוכחי

### לוגיקת Query
כשמשנים תאריך, ה-query יהיה:
```
after:2026/03/07 before:2026/03/08
```
משולב עם חיפוש חופשי אם קיים.
