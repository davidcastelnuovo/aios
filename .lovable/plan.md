

## תוכנית: הצגת ההודעות החדשות ביותר בלבד (דילוג על ישנות)

### מצב נוכחי
התיקון האחרון כבר שינה את הסדר לדילוג על הודעות ישנות (שליפה בסדר יורד עם limit 2000, ואז היפוך). אבל צריך לוודא שזה עובד נכון גם ב-Edge Function `get-chat-history` וגם להוסיף אינדיקציה למשתמש שיש הודעות ישנות שלא מוצגות.

### שינויים נדרשים

1. **עדכון `get-chat-history` Edge Function** - שינוי הסדר גם שם ל-descending + limit, ואז היפוך בתוצאה, כדי שגם קריאות מה-Edge Function יחזירו את ההודעות האחרונות

2. **הוספת אינדיקציה ויזואלית ב-`ChatMessageList.tsx`** - אם מספר ההודעות שהתקבלו שווה ל-limit (2000), להציג באנר בראש הצ'אט שאומר "מציג 2000 הודעות אחרונות בלבד" כדי שהמשתמש ידע שיש עוד הודעות ישנות

3. **וידוא שה-limit עקבי** - להגדיר קבוע `MAX_MESSAGES = 2000` במקום אחד ולהשתמש בו בכל המקומות
