

# תוכנית: בועות הודעה בסגנון WhatsApp לצ'אט הצוותי

## מצב נוכחי
ההודעות מוצגות בסגנון Slack — כולן יושרו לצד אחד, ללא בועות, ללא הבדל צבעוני בין שולחים. קשה מאוד להבדיל בין המשתמשים.

## מה ישתנה

### 1. בועות הודעה (Message Bubbles)
- הודעה **שלי** → בועה ירוקה (כמו WhatsApp), צמודה ל**שמאל** (RTL = שמאל = "ימין" ויזואלית)
- הודעה של **אחרים** → בועה לבנה/אפורה בהירה, צמודה ל**ימין**
- פינות מעוגלות, padding פנימי, צל קל

### 2. צבעי משתמשים
- כל משתמש (שאינו "אני") מקבל צבע ייחודי לשם שלו (מתוך פלטת ~8 צבעים)
- הצבע מחושב מ-hash של ה-`sender_id` — עקבי ויציב

### 3. Layout
- הודעה שלי: `flex-row-reverse` + `items-end` (בועה בצד שמאל)
- הודעה של אחרים: `flex-row` + `items-start` (בועה בצד ימין)
- אווטאר מוצג רק להודעות של אחרים

### 4. מידע meta
- שם השולח מוצג מעל הבועה (רק להודעות של אחרים, רק כשמחליף שולח)
- שעה מוצגת בתוך הבועה בפינה התחתונה

### קבצים לעריכה
- `src/pages/TeamChat.tsx` — שינוי ה-JSX של רינדור הודעות (אזור שורות ~988-1100)

### פרטים טכניים
- פלטת צבעים לשמות: `['#e57373','#64b5f6','#81c784','#ffb74d','#ba68c8','#4db6ac','#f06292','#aed581']`
- פונקציית hash: `senderId.charCodeAt(0) + charCodeAt(1) ... % palette.length`
- בועה שלי: `bg-green-100 dark:bg-green-900/40`
- בועה של אחרים: `bg-white dark:bg-zinc-800 border`
- `max-w-[75%]` לבועות כדי למנוע מתיחה מלאה
